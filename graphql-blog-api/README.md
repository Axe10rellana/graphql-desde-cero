# GraphQL Blog API

## Descripción

Código base del ejemplo GraphQL Blog API donde se aprende a crear una API utilizando GraphQL, Node, Express, MongoDB y Json Web Tokens.

## Comandos

- npm i
- npm run dev
- npm run build

## Dependencias

- Dotenv
- Express
- Express-graphql
- GraphQL
- Json Web Tokens
- Mongoose

## Dependencias De Desarrollo

- Nodemon
